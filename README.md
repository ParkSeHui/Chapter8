#### 느낀점

git이라는 것을 처음 접해보고 배우게 된 시간이었다. 기존에 배우던 다른 언어들은 쉽게 접할 수 있고 익숙해졌지만 git은 생소했다. 
다른 언어에 비한 장점도 많고 분산되어 있는 프로젝트들을 관리하기도 쉽다. 앞으로 git에 관심을 가지고 좀 더 배우고 익힐 수 있는 시간을 가져야겠다.
